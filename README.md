# User Management App

Веб-приложение для управления пользователями с авторизацией, просмотром списка пользователей и детальной информацией о каждом пользователе.
Приложение использует API для получения данных и реализует базовую систему аутентификации с сохранением данных в LocalStorage.

## Функциональность
##### 1. Страница авторизации (Login Page)

Регистрация пользователя с полями: Username, Email, Password, Repeat password. Данные сохраняются в LocalStorage.

Вход в систему по Username и Password.

Валидация: проверка на пустые поля и совпадение паролей.

После входа данные пользователя сохраняются в LocalStorage, происходит редирект на страницу /users.

##### 2. Главная страница (Users Page)

Получение пользователей с API (GET /users).

Отображение карточек пользователей с фото, именем, должностью и городом.

Поиск пользователей по имени.

Фильтрация по полу и департаменту.

Пагинация для удобного просмотра большого количества пользователей.

Отображение состояния загрузки (Loader) и сообщений об ошибке.

Переход на детальную страницу пользователя через кнопку "View Details".

##### 3. Страница пользователя (User Detail Page)

Получение данных пользователя по ID (GET /users/:id).

Отображение фото, имени, возраста, Email, телефона, адреса и университета.

Информация о компании: название, должность, адрес.

Банковская информация: тип карты и срок действия.

Обработка состояния загрузки и ошибок.

Кнопка "Back to Users" для возврата на страницу списка пользователей.

##### 4. Выход из системы (Logout)

Очистка LocalStorage.

Редирект на страницу логина.

## Установка и запуск

1. Клонируйте репозиторий: https://github.com/VikaViktoriaK/User-Managment.git

2. Установите зависимости с помощью npm install.

3. Запустите приложение в режиме разработки с помощью npm start. Приложение будет доступно по адресу http://localhost:8080
.

4. Для сборки проекта используйте npm run build.
